# kernel easy installer
원하는 버전의 커널을 쉽게 설치할 수 있는 bash 스크립트

## 실행방법
"sudo bash update_kernel.sh"를 입력하거나 혹은 su로 root로그인 한 후 "bash update_kernel.sh"입력

## 주의사항
root권한으로 실행해야 정상 실행 가능함
